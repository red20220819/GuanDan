# 掼蛋游戏测试文件

本目录包含用于测试和调试掼蛋游戏各种功能的HTML文件。

## 测试文件说明

### 调试文件
- **debug-bomb.html** - 炸弹牌型调试工具
  - 用于测试各种炸弹牌型的识别和比较
  - 包含可视化的调试界面

### 功能测试文件
- **test-card-removal.html** - 出牌功能测试
  - 测试选牌和出牌逻辑
  - 验证牌型识别准确性

- **test-four-bomb.html** - 四张炸弹测试
  - 专门测试四张炸弹的规则
  - 包含多种炸弹组合测试

- **test-optimized.html** - 性能优化测试
  - 测试游戏性能和加载速度
  - 检查组件加载状态

- **test-suite.html** - 综合测试套件
  - 完整的测试框架
  - 包含所有功能的单元测试和集成测试

## 使用方法

1. 在浏览器中打开对应的测试文件
2. 查看测试结果和调试信息
3. 使用开发者工具查看详细日志

## 注意事项

- 这些测试文件仅供开发调试使用
- 正式游戏请访问 `index.html` 或 `index-modern.html`
- 部分测试文件可能需要特定环境才能正常运行
