# 🎯 掼蛋游戏

## 📋 项目概述

掼蛋游戏 - 采用模块化架构开发的4人扑克游戏，完整实现掼蛋规则。

## 🚀 快速开始

### 启动游戏
```bash
# 方式1：使用npm脚本
npm start

# 方式2：开发模式（自动打开浏览器）
npm run dev

# 方式3：直接启动服务器
node server.js
```

### 访问游戏
- 游戏地址：http://localhost:8080/index-modern.html

## 🃏 游戏规则

**详细规则请查看**：[掼蛋游戏完整规则说明.md](./掼蛋游戏完整规则说明.md)

### 核心规则速览

| 规则项 | 说明 |
|--------|------|
| **牌组** | 2副牌108张，每人27张 |
| **级牌** | 当前打到的级别牌（如打2时，♠2♥2♣2♦2都是级牌，权重101） |
| **逢人配** | 红桃级牌（如♥2），可代任意牌（除王外） |
| **大小顺序** | 大王(103) > 小王(102) > 级牌(101) > A(14) > K > Q > J > 10 > ... > 2 |
| **炸弹等级** | 天王炸弹 > 八炸 > 七炸 > 六炸 > 同花顺 > 五炸 > 四炸 |

### 升级规则
- **头游+二游**：胜方升3级
- **头游+三游**：胜方升2级
- **头游+末游**：胜方升1级
- **打A需双上**才算过关，否则退回J

## 🐛 问题排查

### 常见问题
1. **CORS错误**：使用内置服务器启动游戏
2. **AI不出牌**：查看浏览器控制台错误信息
3. **界面显示异常**：检查CSS文件是否正确加载
