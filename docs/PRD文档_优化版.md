# 掼蛋大师 - 产品需求文档(PRD) - 优化版

## 1. 产品概述

### 1.1 项目背景
掼蛋作为起源于江苏的扑克牌游戏，在全国范围内拥有广泛的群众基础。根据2023年中国棋牌游戏行业报告，掼蛋类游戏的月活跃用户超过3000万，市场规模约15亿元，年增长率保持在20%以上。本项目采用AI辅助开发模式，快速构建专业的网页版掼蛋游戏平台。

### 1.2 市场分析

#### 1.2.1 竞品分析
| 产品名称 | 月活用户 | 核心优势 | 主要劣势 | 市场占有率 |
|---------|---------|----------|----------|------------|
| 欢乐掼蛋 | 800万 | 腾讯流量优势 | 广告过多 | 25% |
| 途游掼蛋 | 500万 | 玩法丰富 | 界面复杂 | 18% |
|  JJ掼蛋 | 300万 | 竞技性强 | 新手门槛高 | 12% |
| 同城游掼蛋 | 200万 | 地方特色 | 画质一般 | 8% |

**竞品分析结论**：
- 市场集中度较低，头部产品优势明显但也有明显短板
- 用户普遍抱怨广告过多、界面复杂、学习成本高
- 存在品质化、轻量化的市场机会

#### 1.2.2 目标用户画像
**核心用户群体（35-55岁）**：
- 地域：江苏、安徽、山东等华东地区占比70%
- 收入：月收入5K-15K的中等收入群体
- 场景：下班后娱乐、朋友聚会、家庭娱乐
- 付费能力：月游戏消费50-200元

**次要用户群体（25-35岁）**：
- 特点：接受新事物能力强，注重游戏体验
- 需求：简洁界面、快速匹配、社交功能
- 付费意愿：为优质体验付费，ARPU值较高

#### 1.2.3 市场规模测算
**TAM（总可用市场）**：15亿元（2023年中国掼蛋游戏市场）
**SAM（服务可用市场）**：4.5亿元（华东地区+部分华中地区）
**SOM（目标市场份额）**：
- 第一年：0.1%（450万元）
- 第三年：0.5%（2250万元）
- 第五年：1.2%（5400万元）

### 1.3 产品定位
- **产品名称**：掼蛋大师
- **产品类型**：网页版掼蛋游戏平台（可移植App）
- **核心特色**：专业掼蛋规则、智能AI对手、优化桌面布局、纯净无广告
- **开发模式**：AI辅助开发，快速迭代
- **技术栈**：Vue 3 + TypeScript + Vite + Figma

- **目标用户**：追求高品质游戏体验的掼蛋爱好者

#### 2.2.2 布局特点说明
**AI玩家显示优化**：
- 只显示3-5张牌背面 + 牌数标签 `[12张]`
- 牌数标签：红色圆形背景，白色数字，醒目显示
- 队友用蓝色边框标识，对手用红色边框标识

**用户手牌显示**：
- 横向排列，部分重叠显示（-20px间距）
- 选中效果：向上位移12px + 金色边框发光
- 支持多选，点击选择/取消选择

**操作按钮区域**：
- 位置：玩家手牌上方，便于操作
- 按钮组：6个核心功能按钮，单行排列
- 按钮设计：图标 + 文字，清晰易懂

**出牌区域设计**：
- 分为三个子区域：上家出牌、队友出牌、对手出牌
- 显示实际牌面，横向重叠排列
- 标注清晰：`上家出牌:`、`队友出牌:`、`对手出牌:`

### 2.3 核心游戏功能

#### 2.3.1 掼蛋规则引擎
- **完整规则**：从2打到A、进贡还贡、升级机制
- **牌型判断**：单张、对子、三张、顺子、同花顺、炸弹、王炸
- **智能出牌**：合法出牌判断、出牌提示
- **胜负判定**：先出完牌的一方获胜

#### 2.3.2 AI对手系统
- **三个难度等级**：
  - 简单（60%胜率）：基础规则遵循，随机策略
  - 中等（75%胜率）：基础记牌，简单配合
  - 困难（85%胜率）：完整记牌，高级策略
- **AI特性**：记牌、算牌、配合队友、智能决策
- **动态调整**：根据玩家胜率自动调整AI难度

### 2.4 用户体验功能

#### 2.4.1 交互体验
- **流畅动画**：发牌、出牌、得分动画效果（<300ms）
- **音效系统**：背景音乐、操作音效（可静音）
- **触觉反馈**：移动端震动反馈
- **响应式设计**：完美适配桌面和移动设备

#### 2.4.2 数据功能
- **用户统计**：胜率、总局数、最高等级、积分排行
- **游戏记录**：历史战绩、详细记录（最近50局）
- **成就系统**：游戏成就、徽章收集
- **设置管理**：音效、画面、游戏偏好设置

- **产品质量**：严格按照优化后的游戏桌面布局实施
